# Research Workflow Contract v1

**Status:** Active research workflow contract
**Current accepted main:** `23b5378 Merge daily coach fully free source-data lab evidence v1`

Research improves deterministic backend design. Research does not directly become provider advice or product truth.

## Research Categories

Approved research categories include exercise science, recovery literature, nutrition research, wearable integrations, health APIs, coaching methodology, provider capabilities, and LLM techniques.

## Core Rule

AI-assisted research informs deterministic backend design. It must not be copied directly into provider advice, medical advice, or user-facing product truth without Architecture review, deterministic modeling, QA evidence, and source documentation.

## Translation Path

```text
research finding
→ architecture interpretation
→ deterministic backend design proposal
→ tests and QA evidence
→ product/user-facing behavior only if accepted
```

## Current Application

Research should support Backend Intelligence Foundation before supporting RAG, vector search, embeddings, multi-agent orchestration, or production-grade provider intelligence.
